# Simple Sharepoint Markdown Edit
Make `.env.local` and define your subdomain.
```
REACT_APP_SHAREPOINT_SUBDOMAIN={your subdomain}
```
Adjust `src/util.js`. After buiding rename `index.html` to `index.aspx`.
